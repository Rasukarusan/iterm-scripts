## Features

- change background color
- change transparency
- set background image
- clear badge
- clear view

## Requirement

- Mac OS
- iTerm

## Install

```sh
git clone https://github.com/Rasukarusan/iterm-scripts.git
```

## Usage

```sh
cd iterm-scripts
sh iterm.sh
```
